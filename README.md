# 🔐 SISS 보안 뉴스 스크랩 스터디

본 저장소는 **SISS 내부 보안 뉴스 스크랩 스터디**를 위한 아카이빙 공간입니다.  
국내·외 보안 뉴스를 기반으로 주요 보안 사고, 취약점(CVE), APT, 악성코드 이슈 등을 정리하고 단순 요약을 넘어 **공격 방식, 의미, 시사점까지 분석**하는 것을 목표로 합니다.

본 활동은 말 그대로 **‘스터디’ 형태**로,별도의 정기 회의 없이 자율적으로 진행되며 각자의 공부와 성장을 위한 기록에 초점을 둡니다.

---

## 📌 스터디 목적
- 최신 보안 이슈 및 트렌드 흐름 파악
- 실제 사고 사례를 통한 공격 기법 이해
- 이론과 실무를 연결하는 사고력 강화
- 학회 차원의 지식 아카이빙 및 공유

---

## 📌 진행 방식
- 국내·외 보안 뉴스 중 자유롭게 주제 선정
- **1~2주에 1회 이상 스크랩 작성 권장**
- 정해진 템플릿 기반으로 정리
- 작성 방식은 아래 중 선택
  - Notion 작성
  - 개인 GitHub에 `md 파일`로 작성 후 본 저장소에 PR / push

※ 세부 규칙은 스터디 진행 상황에 따라 내부 논의를 통해 조정될 수 있습니다.

## 📝 스크랩 작성 가이드

스크랩에는 아래 내용이 포함되기를 권장합니다.

- 사건 개요 및 사실 관
- 공격 방식 및 기술적 요소
- 유출 정보 또는 피해 범위
- 대응 및 후속 조치
- 이슈의 의미 및 시사점
- 추가 학습 포인트

📌 **중요**
- 사실 / 추정 / 분석은 명확히 구분하여 작성
- 추측일 경우 `(추측입니다)` 또는 `(확실하지 않음)` 명시
- 모든 기사 및 자료는 **출처 필수**
